🍇 Acaiteria

Sistema criado para apoiar lojas de açaí iniciantes, oferecendo ferramentas simples de gestão de clientes, produtos e pedidos.

 Objetivo
Dar suporte digital a pequenos negócios de açaí, ajudando na organização e presença online.

 Tecnologias
PHP + MySQL

HTML, CSS, JavaScript

 Estrutura

Código
/src     -> Código-fonte
/assets  -> Imagens e fontes
/db      -> Banco de dados
/docs    -> Documentação
⚙️ Instalação

Clone o repositório:

bash
git clone https://github.com/nainasdc-sketch/acaiteria.git
Importe o banco BACKUP_16062024_sql_acaiteria.sql.

Configure credenciais no PHP e rode em servidor local (XAMPP/WAMP).
